Les informations resterons

possède une communauté d'utilisateurs. Si vous souhaitez traiter les paiements de ces utilisateurs avec Mangopay, ils doivent être créés comme utilisateurs via l'API.

Le traitement des paiements sur votre plateforme est une activité hautement réglementée. Mangopay a l'obligation légale de :

- Établissez l'identité de vos utilisateurs
- Vérifiez leur identité s'ils souhaitent verser de l'argent sur leur compte bancaire

Les obligations des utilisateurs dépendent de leur identité et des actions qu'ils souhaitent effectuer sur votre plateforme. C'est pourquoi Mangopay a créé un système de types et de catégories d'utilisateurs.
