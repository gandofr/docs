## Introduction

La solution de prévention de la fraude de Mangopay vous permet de définir une logique personnalisée pour détecter et atténuer les tentatives de fraude sur votre plateforme.

Avec l' offre Essential , vous pouvez utiliser la solution sans intégration supplémentaire depuis le tableau de bord Mangopay pour surveiller les transactions, créer et tester vos règles, et exploiter les informations réseau de Mangopay. Dans ce cas, la solution s'appuie uniquement sur les informations déjà disponibles dans les données de transaction (telles que l'adresse IP, les informations utilisateur et de paiement, etc.).

Avec l' offre Avancé , vous pouvez également intégrer le profileur à vos pages utilisateurs afin de collecter des données supplémentaires sur leur navigation et d'améliorer vos capacités de détection des fraudes grâce à une technologie propriétaire. Cette option exploite un large éventail de signaux de données permettant de détecter les robots, les VPN, les paramètres réseau anormaux ou les comportements utilisateurs anormaux.
