---

<Warning>
  Pour faire un virement vers son compte bancaire, Il est obligatoire d'avoir passé **la vérification d'identité** et d'avoir **enregistré au moins un compte bancaire**.
  Si ce n'est pas le cas rendez-vous ici pour en savoir plus.
</Warning>

### Effectuer un virement

1. Cliquez sur **Soldes** dans la navigation principale
2. Cliquez sur le bouton **Virer des fonds**
3. Saisissez le montant à transférer (total ou partiel)
4. Choisissez le compte bancaire vers lequel vous souhaitez virer les fonds (si plusieurs comptes enregistrés)
5. Confirmez l’opération

<Info>
  **Information**: Vous ne pouvez pas virer un montant supérieur à votre solde disponible.
</Info>

Les fonds seront transférés sur votre compte bancaire selon les délais que vous avez sélectionnés dans vos paramètres.

![Virerdesfonds Gi](/images/Virerdesfonds.gif)

## Délais de transfert

Dans vos paramètres de compte bancaire (**Profil** **→** **Compte bancaire**), vous avez la possibilité de choisir entre deux délais de virement :

- Les virements standard : Généralement les fonds sont disponibles entre 3 à 5 jours ouvrés.
- Les virements instantanés : Les fonds sont disponibles immédiatement après un transfert.

## Vérifier votre solde Gando

Dans l’interface **Solde**, vous pouvez consulter :

- Le **solde disponible** (transférable immédiatement)
- Le **solde en attente** (fonds en cours de libération)

Ces infos se trouvent dans :

> **Solde → Disponible / En attente**
