</Warning>

## Quels sont les étapes pour ajouter un compte ?

1. Dans la page des Paramètres, allez dans l’onglet **Compte bancaire**
2. Cliquez sur le bouton**Ajouter un compte bancaire**
3. Remplissez les informations demandées :
   - Adresse du compte bancaire
   - IBAN
   - BIC
   - Nom du titulaire du compte bancaire
4. Cliquez sur **Enregistrer**

Vous pouvez supprimer un compte en cliquant sur **Désactiver**.

## Délais de virement

Dans la section **Paramètres \> Compte bancaire** vous pouvez configurer le délai des virements de fonds sur votre compte bancaire.

- Standard : vos fonds sont disponibles sous 3 à 5 jours ouvrés, sans frais.
- Virement instantané : les fonds sont disponibles immédiatement, avec 2€ de frais pour chaque virement.

Ces paramètres s'appliquent à tous vos virements.

## Virements bancaires

Pour virer vos fonds depuis votre compte numérique Gando vers votre compte bancaire, rendez-vous dans la section **Solde** de la navigation principale.

- Cliquez sur le bouton **Virer des fonds**
- Entrez le montant ou cliquez sur **Max** pour virer la totalité des fonds disponibles.
- Choisissez le compte bancaire à créditer.
- Cliquez sur **Virer**

![Popupvirement Pn](/images/Popupvirement.png)

Des frais sont appliqués selon le délai de virement que vous avez configuré dans la section **Paramètres \> Compte bancaire**
