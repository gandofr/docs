## **Si vous ne parvenez pas à vous connecter, après vous être connecté avec succès auparavant**

Actualisez le cache et les cookies de votre navigateur. Nous vous recommandons d'utiliser un ordinateur de bureau pour [<u>vous connecter</u>](https://platform.openai.com/login) .

**Si vous voyez « Il existe déjà un utilisateur avec une adresse e-mail... » ou « Méthode d'authentification incorrecte »...**

- Vous verrez cette erreur si vous tentez de vous connecter avec une méthode d'authentification différente de celle utilisée lors de l'inscription initiale. Votre compte ne peut être authentifié que si vous vous connectez avec la méthode d'authentification utilisée lors de l'inscription initiale. Par exemple, si vous vous êtes inscrit avec Google Sign-in, veuillez continuer à utiliser la même méthode.
- Si vous n'êtes pas sûr de la méthode que vous avez utilisée à l'origine pour vous inscrire, essayez [<u>de vous connecter</u>](https://platform.openai.com/login) avec chacune des méthodes suivantes à partir d'une fenêtre de navigation privée autre que Firefox :
  - Nom d'utilisateur \+ mot de passe

**Si vous essayez de vous inscrire et que vous voyez « Cet utilisateur existe déjà »...**

- Cela signifie probablement que vous avez déjà commencé le processus d'inscription, mais que vous ne l'avez pas terminé. Essayez plutôt de [<u>vous connecter .</u>](https://platform.openai.com/login)

**Si vous recevez toujours des erreurs « Une erreur s'est produite » ou « Oups… », essayez ce qui suit :**﻿

1. Actualisez votre cache et vos cookies, puis essayez de vous connecter avec la méthode d'authentification choisie.
2. Essayez une fenêtre de navigateur incognito pour terminer la connexion
3. Essayez de vous connecter à partir d'un autre navigateur/ordinateur pour voir si le problème persiste, car un module complémentaire ou une extension de sécurité peut parfois provoquer ce type d'erreur.
4. Essayez un autre réseau (connexion filaire, Wi-Fi domestique, Wi-Fi professionnel, Wi-Fi bibliothèque/café et/ou réseau cellulaire).

Il est également possible que cette erreur soit due à [<u>la vérification du domaine</u>](https://help.openai.com/en/articles/8871611-domain-verification#h_228a0400b1) . Vous pouvez contacter votre administrateur informatique pour savoir si une autre organisation OpenAI a déjà vérifié votre domaine.
